# Moviechase

## Stack

Moviechase is a Front end website built with Next.js and TypeScript, styled with Tailwind CSS. All data used in this project comes from TMDB [The movie database API](https://developer.themoviedb.org/reference/intro/getting-started)

## What is about?

The goal of this website is to provide users with an engaging platform for discovering movies, TV shows, and celebrities, all while ensuring a perfectly clean and intuitive user experience (UX) and user interface (UI). By prioritizing simplicity and clarity in design, we aim to make navigation seamless and enjoyable, allowing users to effortlessly explore a wide array of content.
